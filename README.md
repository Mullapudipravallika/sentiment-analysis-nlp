# Sentiment Analysis using NLP

This project implements a simple sentiment analysis system that classifies text as positive or negative using Natural Language Processing (NLP) techniques.

## Overview
The model uses TF-IDF vectorization for feature extraction and Logistic Regression for classification.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- TF-IDF Vectorizer

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python sentiment_analysis.py

## Results
The model prints classification accuracy on test data.
